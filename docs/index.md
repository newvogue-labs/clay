# Clay — Reference Docs

Публичная справочная документация системы **Clay** — human-in-the-loop
торговой системы (Binance) с принципом «backend = source of truth».

Сайт собирается из каталога `docs/` репозитория и содержит **актуальную**
архитектурную документацию: решения (ADR), обзор системы и операционные
руководства. Историко-плановые материалы в сайт не включены.

## Разделы

- **Decisions (ADR)** — актуальные архитектурные решения (`016+`).
- **Overview** — blueprint и утверждённый стек.
- **Runbooks** — операционные руководства и деплой.

!!! note "Статус"
    Документация отражает текущее состояние `main`. Знание-вики (vault) —
    advisory-only и не является control-path (ADR-030).
