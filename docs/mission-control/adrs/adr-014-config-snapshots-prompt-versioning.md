# ADR-014: config_snapshots — версионирование промптов ролей

- **Status:** Proposed (реализация — отдельная волна)
- **Date:** 2026-06-13
- **Связь:** готовит Фазу 3 AI-панели UI (чат-окно + редактирование поведения роли).

## Контекст

`role_prompts` живут в коде (`lifespan.py`). Промпт роли = торгово-критичный артефакт;
правка из UI требует версионирования и governance.

## Решение

Переезд промптов в DB с версионированием + governance review→apply (по аналогии с
assignment review→apply). Промпт нельзя менять «вживую» без аудита.

## Последствия

- (+) Промпты редактируемы/аудируемы без деплоя кода.
- (+) Снимает блокер Фазы 3 UI.
- (−) Новый governance-путь + миграция; до реализации промпты остаются в коде.
