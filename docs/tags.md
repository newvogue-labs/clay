# Tags

Domain taxonomy for Clay reference docs. Each tag groups pages by subject area.

## Controlled vocabulary

| Tag | Scope |
|-----|-------|
| **architecture** | System design, C4 diagrams, module maps |
| **execution** | Trade execution, order lifecycle, replay |
| **risk** | Position sizing, exposure limits, guardrails |
| **signals** | Signal pipeline, freshness, quality gates |
| **knowledge** | #knowledge vault, advisory cards, Notion mirror |
| **llm** | LLM roles, model config, egress gateway |
| **infra** | Systemd, pool health, config management |
| **ops** | Retention, deploy runbooks, operations |

:material-tags:

Tags are assigned via YAML frontmatter on each page.
